# Fancify My Text

## Files
- fancifymytext.html - main webpage
- fancifymytext.js - JavaScript file

## Features
- Bigger button (increases font size)
- FancyShmancy (bold, blue, underline)
- BoringBetty (normal text)
- Moo button (uppercase + "-MOO")

## How to Run
Open `fancifymytext.html` in a browser/live server.
